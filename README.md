An app that cna scans inputed information to see analytics of various varibales.

--MVP (Minimum Viable Product)

- Column Data
  - title
  - Company
  - Date
- see graphs and charts
  - pie charts
  - bar charts
  - line graphs
- Can edit the existing data
- column that says whether you are waiting, got ghosted, rejected, in progress, success

Features

- comparing up to 4 graphs all with different variables
- Ability to add new variables to the Column Data
- default percentage
  - the percentage ghosted
  - amount of application (yearly, monthly, weekly, all time)
  - percentage rejected
  - got interviews
